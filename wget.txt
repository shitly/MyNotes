有时候我们需要wget一个文件下载到指定的目录下，或者重命名成指定的名字
wget -r -p -np -k -P ~/tmp/ http://Java-er.com

wget居然有如此邪恶的用法

这个语句表示下载全站资料

等于把一个网站整个静态化到本地。


解释一下参数

-P 表示下载到哪个目录
-r 表示递归下载
-np 表示不下载旁站连接.
-k 表示将下载的网页里的链接修改为本地链接.
-p 获得所有显示网页所需的元素

额外的
-c 断点续传
-nd 递归下载时不创建一层一层的目录，把所有的文件下载到当前目录
-L 递归时不进入其它主机，如wget -c -r www.xxx.org/
-A 指定要下载的文件样式列表，多个样式用逗号分隔
-i 后面跟一个文件，文件内指明要下载的URL

* 使用代理下载
wget -Y on -p -k https://sourceforge.NET/projects/wvware/

代理可以在环境变量或wgetrc文件中设定

# 在环境变量中设定代理
export PROXY=http://211.90.168.94:8080/
# 在~/.wgetrc中设定代理
http_proxy = http://proxy.yoyodyne.com:18023/
ftp_proxy = http://proxy.yoyodyne.com:18023/

wget各种选项分类列表

* 启动

-V, Cversion 显示wget的版本后退出
-h, Chelp 打印语法帮助
-b, Cbackground 启动后转入后台执行
-e, Cexecute=COMMAND
执行`.wgetrc’格式的命令，wgetrc格式参见/etc/wgetrc或~/.wgetrc

* 记录和输入文件

-O, Coutput-file=FILE 把记录写到FILE文件中
-a, Cappend-output=FILE 把记录追加到FILE文件中
-d, Cdebug 打印调试输出
-q, Cquiet 安静模式(没有输出)
-v, Cverbose 冗长模式(这是缺省设置)
-nv, Cnon-verbose 关掉冗长模式，但不是安静模式
-i, Cinput-file=FILE 下载在FILE文件中出现的URLs
-F, Cforce-html 把输入文件当作HTML格式文件对待
-B, Cbase=URL 将URL作为在-F -i参数指定的文件中出现的相对链接的前缀
Csslcertfile=FILE 可选客户端证书
Csslcertkey=KEYFILE 可选客户端证书的KEYFILE
Cegd-file=FILE 指定EGD socket的文件名

* 下载

Cbind-address=ADDRESS
指定本地使用地址(主机名或IP，当本地有多个IP或名字时使用)
-t, Ctries=NUMBER 设定最大尝试链接次数(0 表示无限制).
-O Coutput-document=FILE 把文档写到FILE文件中
-nc, Cno-clobber 不要覆盖存在的文件或使用.#前缀
-c, Ccontinue 接着下载没下载完的文件
Cprogress=TYPE 设定进程条标记
-N, Ctimestamping 不要重新下载文件除非比本地文件新
-S, Cserver-response 打印服务器的回应
Cspider 不下载任何东西
-T, Ctimeout=SECONDS 设定响应超时的秒数
-w, Cwait=SECONDS 两次尝试之间间隔SECONDS秒
Cwaitretry=SECONDS 在重新链接之间等待1…SECONDS秒
Crandom-wait 在下载之间等待0…2*WAIT秒
-Y, Cproxy=on/off 打开或关闭代理
-Q, Cquota=NUMBER 设置下载的容量限制
Climit-rate=RATE 限定下载输率

* 目录

-nd Cno-directories 不创建目录
-x, Cforce-directories 强制创建目录
-nH, Cno-host-directories 不创建主机目录
-P, Cdirectory-prefix=PREFIX 将文件保存到目录 PREFIX/…
Ccut-dirs=NUMBER 忽略 NUMBER层远程目录

* HTTP 选项

Chttp-user=USER 设定HTTP用户名为 USER.
Chttp-passwd=PASS 设定http密码为 PASS.
-C, Ccache=on/off 允许/不允许服务器端的数据缓存 (一般情况下允许).
-E, Chtml-extension 将所有text/html文档以.html扩展名保存
Cignore-length 忽略 `Content-Length’头域
Cheader=STRING 在headers中插入字符串 STRING
Cproxy-user=USER 设定代理的用户名为 USER
Cproxy-passwd=PASS 设定代理的密码为 PASS
Creferer=URL 在HTTP请求中包含 `Referer: URL’头
-s, Csave-headers 保存HTTP头到文件
-U, Cuser-agent=AGENT 设定代理的名称为 AGENT而不是 Wget/VERSION.
Cno-http-keep-alive 关闭 HTTP活动链接 (永远链接).
Ccookies=off 不使用 cookies.
Cload-cookies=FILE 在开始会话前从文件 FILE中加载cookie
Csave-cookies=FILE 在会话结束后将 cookies保存到 FILE文件中

* FTP 选项

-nr, Cdont-remove-listing 不移走 `.listing’文件
-g, Cglob=on/off 打开或关闭文件名的 globbing机制
Cpassive-ftp 使用被动传输模式 (缺省值).
Cactive-ftp 使用主动传输模式
Cretr-symlinks 在递归的时候，将链接指向文件(而不是目录)

* 递归下载

-r, Crecursive 递归下载－－慎用!
-l, Clevel=NUMBER 最大递归深度 (inf 或 0 代表无穷).
Cdelete-after 在现在完毕后局部删除文件
-k, Cconvert-links 转换非相对链接为相对链接
-K, Cbackup-converted 在转换文件X之前，将之备份为 X.orig
-m, Cmirror 等价于 -r -N -l inf -nr.
-p, Cpage-requisites 下载显示HTML文件的所有图片

* 递归下载中的包含和不包含(accept/reject)

-A, Caccept=LIST 分号分隔的被接受扩展名的列表
-R, Creject=LIST 分号分隔的不被接受的扩展名的列表
-D, Cdomains=LIST 分号分隔的被接受域的列表
Cexclude-domains=LIST 分号分隔的不被接受的域的列表
Cfollow-ftp 跟踪HTML文档中的FTP链接
Cfollow-tags=LIST 分号分隔的被跟踪的HTML标签的列表
-G, Cignore-tags=LIST 分号分隔的被忽略的HTML标签的列表
-H, Cspan-hosts 当递归时转到外部主机
-L, Crelative 仅仅跟踪相对链接
-I, Cinclude-directories=LIST 允许目录的列表
-X, Cexclude-directories=LIST 不被包含目录的列表
-np, Cno-parent 不要追溯到父目录
